# Privacy Policy

_Last updated: 2026-04-05_

DevPing is a local macOS utility that helps you notice when supported AI coding tools finish or need your attention.

## What DevPing stores locally

DevPing stores local preferences on your Mac using macOS user defaults, including settings like:

- selected sounds
- popup display preferences
- timing preferences
- appearance settings
- onboarding completion state

## What DevPing changes

When you install DevPing integration for a supported tool, it may update local config files on your machine so those tools can call DevPing when they finish or need approval.

See [`WHAT-DEVPING-CHANGES.md`](./WHAT-DEVPING-CHANGES.md) for the exact file types and locations.

## What DevPing sends over the network

The core DevPing app does not require an account and does not need to send your coding session contents to a DevPing server to work.

Links inside the app may open external websites such as:

- GitHub
- community pages
- social profiles
- support pages

## What DevPing does not collect by default

DevPing does not exist to read your prompts, code, or terminal contents for cloud processing.

By default, the launch-ready core product should not collect:

- your source code
- your prompts
- your session transcripts
- your clipboard
- your keystrokes

## Future paid features

If DevPing later adds optional paid accounts, licenses, sync, or product updates, this policy should be updated before those features are enabled.
