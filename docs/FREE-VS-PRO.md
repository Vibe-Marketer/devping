# Free vs Pro

## Launch recommendation

Ship DevPing free first.

Do not block launch on licensing, accounts, or a payment backend.

## Free
The free version should be genuinely useful.

Suggested free scope:

- Claude Code support
- OpenCode support
- Aider support
- native notifications
- sound selection
- popup timing and positioning
- menu bar controls
- onboarding
- core appearance settings

## Pro (later)
Charge for expanded workflow value, not for the basic notifier itself.

Suggested Pro ideas:

- advanced themes / premium theme packs
- per-tool notification profiles
- advanced routing / rule engine
- richer schedules and DND presets
- notification history
- export/import settings
- team presets
- optional sync

## Recommended pricing direction

### Option A — one-time
- $19–39 one-time upgrade

### Option B — optional subscription later
Only if cloud/sync/team features are added.

## What not to do before launch
- mandatory login
- mandatory account creation
- paywall the core notifier
- build a licensing backend before shipping the free product
